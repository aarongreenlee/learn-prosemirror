Editor Options

Editor options define the general properties of the editor, similar to the way an HTML element's attributes work.

Options are defined in the object passed in the editor constructor.

Placing the editor

The place option describes which DOM element the editor will be placed.

import {ProseMirror} from "prosemirror/dist/edit"

let pm = window.pm = new ProseMirror({
  place: document.querySelector("#editor-1")
})

Setting the initial editor content

The doc option sets the editor's initial content.

import {ProseMirror} from "prosemirror/dist/edit"
import {fromHTML} from "prosemirror/dist/format"
import {defaultSchema} from "prosemirror/dist/model"

let pm2 = window.pm2 = new ProseMirror({
  place: document.querySelector("#editor-2"),
  doc: fromHTML( defaultSchema, "<p>Here's an <em>editor</em> with <b>default content</b>.</p>" )
})

Notice we're including a few functions from ProseMirror modules to convert HTML text to a ProseMirror document node object. fromHTML() requires a document schema, which we'll get into a bit later.

fromDOM() is another function in the format module that accepts a DOM element and converts it into a document node.

If you're looking to initialize a ProseMirror document from a different format, you could write a parser which outputs a ProseMirror document.

Including menus and auto-input rules

You can add functionality to stuff by importing modules and setting the editor options appropriately.

Here, we include the modules for the menu bar, the tooltip (contextual) menu, and auto input rules.

import {ProseMirror} from "prosemirror/dist/edit"
import {fromHTML} from "prosemirror/dist/format"
import {defaultSchema} from "prosemirror/dist/model"
import "prosemirror/dist/inputrules/autoinput"
import "prosemirror/dist/menu/tooltipmenu"
import "prosemirror/dist/menu/menubar"

let pm3 = window.pm3 = new ProseMirror({
  place: document.querySelector("#editor-3"),
  doc: fromHTML( defaultSchema, "<p>Here's an <em>editor</em> with <b>default content</b>. <em>Highlighting text brings up the contextual menu</em>.</p><p>Try entering <code>*+Space</code> at the beginning of a line to see autoinput rules in action.</p>" ),
  menuBar: true,
  tooltipMenu: true,
  autoInput: true
})
